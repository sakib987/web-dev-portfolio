Vanilla JavaScript

Vanilla JavaScript refers to plain, native JavaScript without any libraries or frameworks. It is used to create interactive and dynamic features directly in websites. Learning Vanilla JavaScript gives you a strong foundation in core JavaScript concepts and browser manipulation.

React

React is a JavaScript library developed by Facebook for building user interfaces (UIs). It uses a component-based architecture that makes code reusable and organized. React is widely used to build single-page applications (SPA) with fast performance and dynamic content.

Node Js

Node.js is a JavaScript runtime environment that allows you to run JavaScript on the server side. It helps to build fast and scalable backend applications. Powered by Google's V8 engine, Node.js can handle multiple requests efficiently and is great for real-time apps.

MongoDB

MongoDB is a popular NoSQL database where data is stored in document format (JSON-like). It is schema-less, making it flexible and easy to manage data. MongoDB is widely used for modern applications because of its speed, scalability, and high performance.
